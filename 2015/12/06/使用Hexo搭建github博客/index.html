使用Hexo搭建github博客

发表于 2015-12-06   |   分类于 hexo

使用环境

Git
Github仓库:xxx.github.io
hexo 3.x

安装Hexo

npm install -g hexo

初始化blog框架

//创建博客文件夹
mkdir GitBlog
cd GitBlog

//初始化    
hexo init
npm install
npm install hexo-deployer-git --save

修改_config.yml

使用ssh地址

部署启动

生成静态页面，自带一个helloworld.md

//hexo generate
hexo g

部署到git

//hexo deply
hexo d

也可以直接使用

heo d -g

本地预览,启动后在http://localhost:4000/中查看

//hexo server
hexo s

写blog

将写好的md文件放到source/_posts下，在开头添加一些描述信息，分类和标签不是必须的。

或者使用命令hexo n,将会生成一个md文件，直接将你的md文件追加进去就行了

hexo n 文章名

将更改部署到git

hexo d -g
